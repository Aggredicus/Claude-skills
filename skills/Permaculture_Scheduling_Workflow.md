# Permaculture Scheduling & Inventory Workflow

## Pipeline

1.  Consolidate sources and deduplicate species.
2.  Add agronomic metadata (germination, harvest, spacing).
3.  Derive planting windows (Michigan defaults unless specified).
4.  Build succession calculator.
5.  Export Google Sheets-ready XLSX.

## Safeguards

-   Separate direct sow vs transplant.
-   Flag cultivar ambiguity.
-   Document data sources.
