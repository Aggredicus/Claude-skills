# Universal Quality Gates

## Gate 1 --- Assumptions Declared

All inferred parameters are explicitly labeled.

## Gate 2 --- Consistency Checks

Units, timestamps, IDs, and versions must not drift.

## Gate 3 --- Failure Modes

Artifacts must include: - Input validation - Error messaging - Safe
reset state

## Gate 4 --- Future-You Friendliness

Each artifact includes usage instructions and example inputs.
